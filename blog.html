Important FAQ

What is Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where:
1. HTML elements are nodes in the tree.
2. Developers can use JavaScript to dynamically access, modify, add, or delete elements and their content or attributes.

How do you select an element from DOM?

You can select an element from the DOM using JavaScript methods like:

1. document.getElementById("id"): Selects an element by its ID.
2. document.querySelector("selector"): Selects the first element matching a CSS selector.
3. querySelectorAll("selector"): Selects all elements matching a CSS selector (returns a NodeList).
4. document.getElementsByClassName("class"): Selects elements by class name (returns an HTMLCollection).
5. document.getElementsByTagName("tag"): Selects elements by tag name (returns an HTMLCollection).

What is event delegation in the concept of DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements, even if they are dynamically added later.

Why it's useful:
1. Improves performance: Reduces the number of event listeners by attaching one to the parent.
2. Handles dynamic elements: Captures events on child elements that are created after the event listener is added.
3. Simplifies code: Avoids attaching separate listeners to multiple child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM as follows:

Attributes :

1. Get an attribute: element.getAttribute("attributeName")
2. Set an attribute: element.setAttribute("attributeName", "value")
3. Remove an attribute: element.removeAttribute("attributeName")

Styles :

1. Access styles: element.style.property = "value" (e.g., element.style.color = "red";)
2. Add multiple styles: Use element.setAttribute("style", "property:value; property2:value;");
3. Use CSS classes: element.classList.add("className"), element.classList.remove("className"), or element.classList.toggle("className").
